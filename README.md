# Golar

Golar is an embedded language tooling orchestrator based on [typescript-go](https://github.com/microsoft/typescript-go). It enables type checking, linting, and LSP support for TypeScript-based languages like Astro, Svelte, and Vue through typescript-go. Its architecture is inspired by [@johnsoncodehk](https://github.com/johnsoncodehk)'s [Volar.js](https://github.com/volarjs/volar.js).

## Getting Started

Read the [getting started guide](https://golar.dev/guides/getting-started/).

## Language support

Currently, Golar supports Astro, Ember, Svelte, and Vue by integrating their official language tooling. Read the docs to learn how to configure Golar to work with them:

- [Astro](https://golar.dev/languages/astro/)
- [Ember](https://golar.dev/languages/ember/)
- [Svelte](https://golar.dev/languages/svelte/)
- [Vue](https://golar.dev/languages/vue/)

> [!NOTE]
> Golar is currently in active development. Expect breaking changes.
>
> You can read more about Golar's roadmap in ["Golar Gets a Linter. What's Next?"](https://golar.dev/blog/golar-gets-a-linter/)
