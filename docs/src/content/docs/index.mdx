## Why Golar

<CardGrid>
	<Card title="One command by default">
		Run linting and type-checking together with a single `golar` command and
		reuse compiler work across both steps.
	</Card>
	<Card title="Plugin-based language support">
		Add support for Astro, Ember, Svelte, and Vue through language plugins
		instead of maintaining separate CLI flows.
	</Card>
	<Card title="Type-aware custom rules">
		Build custom lint rules in JavaScript or Rust and query type information
		directly from typescript-go.
	</Card>
</CardGrid>
